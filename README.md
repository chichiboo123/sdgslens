# SDGs 렌즈 (SDGs Lens)

세계 뉴스를 **SDGs 17개 목표 관점으로 읽고 해석하는 교육용 웹앱**입니다.  
UN·UNICEF·UNDP 계열 기사 데이터를 바탕으로, 학습자가 직접 목표를 선택하고 피드백을 받으며 세계시민 감수성을 키울 수 있도록 설계되었습니다.

**라이브 서비스**: [sdgslens.chichiboo.link](https://sdgslens.chichiboo.link)

---

## 1) 앱 개요

- **프로젝트명**: SDGs 렌즈
- **형태**: 별도 빌드 도구 없이 실행 가능한 정적 웹앱 (HTML/CSS/Vanilla JS)
- **배포**: GitHub Pages (커스텀 도메인: sdgslens.chichiboo.link)
- **핵심 목적**: 
  - 국제 이슈를 SDGs와 연결해 해석하는 사고력 강화
  - 어린이·청소년의 세계시민 교육 실천
  - 뉴스 읽기 + SDGs 분류 + 생활 연결 글쓰기의 통합 학습 경험 제공

---

## 2) 주요 기능

### ✅ 뉴스 기반 SDGs 분류 학습
- "뉴스 가져오기" 버튼을 누르면 데이터셋(20건)에서 뉴스 1건을 랜덤으로 로드합니다.
- 학습자는 기사와 태그를 읽고 관련 SDGs를 복수 선택할 수 있습니다.
- 이미 본 뉴스는 `used` 배열로 관리하여 전체 소진 전 중복 노출을 줄입니다.

### ✅ 3단계 학습 흐름 UI
1. 뉴스 보기
2. SDGs 고르기
3. 분석 결과 확인

학습 진행 상태가 상단 스텝 인디케이터로 시각화됩니다.

### ✅ 힌트/재도전/다음 뉴스
- 힌트 토글로 기사 해석 도움 제공
- 같은 기사 재도전(다시 풀기) 가능
- 다음 뉴스로 연속 탐구 가능

### ✅ 자동 채점 + 설명 피드백
- 정답 SDGs와 사용자 선택을 비교해 정답률(%) 산출
- 정답 정확도에 따라 격려 문구가 다르게 출력
- 정답 SDG 칩과 누락 항목을 시각적으로 표시

### ✅ 생활 연결 글쓰기
- "우리 생활과 어떻게 연결되는지"를 직접 작성
- 결과 영역에서 학습자 작성문과 확장 설명을 함께 제시
- 행동 미션(오늘의 실천)까지 제공

### ✅ 탐구 기록 PNG 내보내기
- 분석 완료 후 탐구 결과를 이미지(PNG)로 저장하거나 클립보드에 복사할 수 있습니다.
- 내보내기 카드에는 뉴스 기사·선택한 SDGs·나의 생각이 함께 담깁니다.
- 학습 포트폴리오, 모둠 발표, SNS 공유에 바로 활용 가능합니다.

### ✅ 탐구 완료 카운터
- 헤더 배지에 세션 중 완료한 탐구 횟수를 실시간으로 표시합니다.

### ✅ 다국어 지원 (KO/EN/JA)
- 한국어/영어/일본어 전환 지원
- 언어 선택은 `localStorage`에 저장되어 재방문 시 유지
- 뉴스 텍스트·버튼·도움말·사전 모두 언어별로 현지화
- 좁은 화면(480px 이하)에서는 언어 버튼이 컴팩트 팝업 드롭다운으로 전환됩니다.

### ✅ SDGs 사전 모달
- 목표별 설명, 핵심 타깃, 출처 정보를 모달로 제공
- 뉴스 학습 전/중 참고 가능한 온디맨드 학습 보조 도구

## 3) 기술 구성

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **외부 리소스**:
  - Google Material Symbols
  - Pretendard 폰트(CDN)
- **데이터 구조**:
  - `NEWS_DB`: 뉴스 본문/힌트/피드백/정답 SDG/원문 링크/다국어 텍스트 (20건)
  - `I18N`: UI 전반 다국어 사전
  - `SDG_DICT`: SDGs 설명형 학습 사전

---

## 4) 로컬 실행 방법

아래 중 하나로 실행하면 됩니다.

### 방법 A) 파일 직접 열기
1. 저장소를 내려받습니다.
2. `index.html`을 브라우저로 엽니다.

### 방법 B) 로컬 서버 실행(권장)
```bash
python3 -m http.server 8000
```
브라우저에서 `http://localhost:8000` 접속

## 6) 교육적 활용 제안

- **교실 활동**: 모둠별로 같은 기사에 대해 SDG 선택 비교/토론
- **글쓰기 확장**: "생활 연결" 문장을 행동계획서로 발전
- **포트폴리오**: PNG 내보내기로 탐구 기록을 학습 포트폴리오에 첨부
- **융합 수업**: 사회·도덕·영어·창체(세계시민교육) 연계
- **형성평가**: 정답률보다 '근거 설명' 중심 루브릭 평가 권장

## 8) 원본 자료 소스에 대한 리스펙 (Respect)

이 앱은 국제기구의 공익적 정보 공개와 저널리즘 가치 위에서 학습 목적으로 구성되었습니다.  
원문을 제공해 준 UN/UNICEF/UNDP 및 관련 보고·브리핑 제작자들에게 깊이 감사드립니다.

- UN News
- UNICEF
- UNDP
- (일부 기사 맥락상) FAO, WHO, UNEP, UNESCO 등 UN 체계 기관/협력기구

> 본 프로젝트는 원문을 대체하지 않으며, 학습자가 **반드시 원문 링크를 통해 1차 출처를 확인**하도록 권장합니다.
